Это первый обучающий проект "для себя" с технологиям laravel + vue.js.
Логика личного кабинета, просмотра списка постов, сортировки, фильтрации, пагинации сделана с помощью laravel.
Пользовательский интерфейс админки для управления постами, категориями и пользователями разработаны с использованием vue.js
Также опробовано взаимодействие со сторонними API - сервис погоды.
Привлекаются сторонние библиотеки - phpmailer, jquery, fancybox, axios и др.
Используется препроцессор SASS и сборщик модулей js - Vite.

Данные для входа в панель администратора:
логин: dimas
пароль: 1Aa$$$# 

Дамп базы данных - my_base.sql

Внешнему виду приложения не уделялось внимания. Основной целью было опробовать новые технологии.
